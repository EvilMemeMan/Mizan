# Mizan — ميزان
## Egypt Economic Intelligence

Live at: `https://YOUR_USERNAME.github.io/mizan`

---

### What this is
A 5-tab economic intelligence dashboard tracking Egypt's economy against 10 development models — with live news, reform tracking, regulatory diff view, economic calendar, and model alignment scoring.

### Tabs
- 📰 **News** — Live briefings from Egyptian + international sources
- 📊 **Models** — 10 economic models with dynamic alignment percentages
- 🗓 **Calendar** — IMF reviews, CBE meetings, parliamentary sessions
- 📋 **Reforms** — Lifecycle tracker (Announced → Enacted → Stalled)
- ⚖ **Regulations** — Arabic-sourced regulatory framework with before/after diff

### Updating the app
1. Edit `index.html` with the new version
2. Commit and push to `main`
3. GitHub Actions automatically redeploys in ~30 seconds
4. Your live URL updates instantly

### Usage on phone
1. Open `https://YOUR_USERNAME.github.io/mizan` in Chrome
2. Tap ⋮ → Add to Home Screen
3. Enter your Anthropic API key in ⚙ settings
